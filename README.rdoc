== README

This is a recipe web application. It allows one to create a new recipe with an accompanying description and instructions describing how to prepare the dish. There is image upload support, where the final image of the food can be uploaded and will be displayed in the home page.
The app also has login support, where others users can create an account and be able to comment on recipes. 



Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
